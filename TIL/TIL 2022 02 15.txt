메타 데이터 요소

메타 데이터란 ? 데이터를 설명하는 데이터

만약 책이다 하면 책의 저자, 발간일, 제목, 출판사 같은것들

title 크롬 상단 탭에있는것

meta 데이터 name content 특성을 제공

application-name : 고정된 이름
ex.) 네이버 페이스북 인스타그램

title : 페이지별 대표이름
ex.) 네이버-스포츠, 페이스북-메세지

charset : 문자 인코딩 "UTF-8" 모든 세계언어 지원 유니코드

viewport > name으로 들어감.

link 하이퍼링크를 헤드에도 써 넣음
ex.) css파일 연결

mime 타입 파일 경로를 해석하지 못할 경우 설명해주는 것. 거의 모든 파일에 대한 확장자가 있기 때문에 어떻게 작성하는지만 알면 됨.
ex.) text/css, image/gif

style : html 내부에서 스타일 요소를 지정해서 만들 수도 있음. inline 요소 head에서 써넣는거임.
태그가 아닌 속성으로 쓰면 그 코드만 적용

script : 인라인 or 외부에서 연결하기위해서
보통 자바s를 연결할 때
헤드에 있어도 되고 바디에 있어도 됨.
근데 보통 스크립트가 올라가야 화면창에 로딩이 뜨기 때문에 스크립트는 보통 바디 뒤에 써 놓음.

전역 속성
id : 문서 전체에서 고유식별자라고 함
하나의 요소에 하나의 단어 공백작성 X
구분할거면 "-", " _", " ' "
class : 문서에서 접근자라고 함
id는 한개만 class는 여러개 가능 공백 사용가능
그러나 공백을 쓴다는건 또 다른 class를 추가하는것
단어 자체에 스페이스를 쓰고싶으면 똑같이 -_사용

style 속성 : 코드 하나만 css형식으로 꾸미기
title 속성 : 말풍선을 띄우는 것
lang : 진짜로 언어 vs meta charset utf-8은 컴퓨터가 이해하는 언어
data - ~~~ : data - 뒤에 내가 지정하는 데이터 속성
draggable : 마우스로 끌 수 있는가? t or f 명시해야됨
hidden : 무언가를 숨길 때 사용
