임베디드 요소
( 내용을 코드로 쓰지 않고 외부에서 불러옴)

img : 빈요소
(절대경로) (상대경로)
src 속성 : sorce라는 뜻 url, 이미지가 위치한 경로
alt 속성 (alternative text)
이미지에 대한 설명을 텍스트로 입력
height : 높이 (픽셀 기준 고유 높이 단위 없는 정수)
width : 가로 (픽셀 기준 고유 너비 단위 없는 정수)
srcset : 이미지의 URL (반응형 이미지)
큰 화면엔 큰 그림 작은 화면엔 작은 그림
sizes : 특정 조건에 따라 아예 크기를 지정
video : 자식요소 가능 태그 사이에 재생이 안될 경우 안된다는 메세지를 작성해야됨
src 속성 <source>태그 둘다 사용 가능.
autoplay 속성 : 알아서 자동 실행
(f5하면 정지됨 : 새 페이지 들어가면 갑자기 소리나는게 부정적인 경험이 될 수 있어서. 소리를 제거하면 계속 적용됨)
muted 속성 : 소리 제거
controls 속성 : 비디오 플레이어 적용
loop : 계속 반복 재생
poster : 썸네일 같은 개념 (이게 없으면 첫화면이 썸네일임)
audio : video 태그와 거의 비슷하게 속성을 가짐
canvas : 그래픽 그림 등을 그리기 위한 마크업 요소 뭔가 그리거나 하고싶으면 자바스크립트
iframe : 프레임 안에 웹사이트를 띄우는 형태로 만듬. 구글 지도 같은 것들 넣는것

웹 이미지 유형
래스터 이미지
> 확대할 수록 깨짐
> 모눈을 하나 하나 채우는 형식으로 그림
> 사진 동영상
jpeg : 정지 이미지 압축률이 좋아 빠르게 띄움
png : 요즘 많아짐 jpeg보다 좋음 투명도 적용됨.
gif : 움짤 > 애니매이션 표현
webp : 구글에서 만듬 위에 3개보다 좋은거
백터 이미지
> 확대해도 안 깨짐
> 직선 하나를 쭉 이어서 그려짐
> 다양한 크기로 그려야 하는 아이콘 다이어그램
svg : 안깨지는 그림파일
